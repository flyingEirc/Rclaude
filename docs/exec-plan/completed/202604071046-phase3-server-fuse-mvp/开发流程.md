### 过程中的实际问题

- `make fmt` 无法执行：当前 PowerShell 环境不存在 `make` 命令
- `gci` 在 PowerShell 中命中别名 `Get-ChildItem`，需要显式使用 `gci.exe`
- `golangci-lint` 首轮暴露：
  - `pkg/session.Request` 复杂度超阈
  - `app/server.runServer` 复杂度超阈
  - `internal/testutil.MockConnectStream.RecvMsg` 复制 proto 结构触发 `copylocks`
  - `pkg/fusefs` 的 Linux 专属辅助函数在 Windows 下被判定为 unused
- 上述问题均已通过拆函数、改用 `proto.Merge`、把 Linux 专属辅助函数移动到 `mount_linux.go` 等方式修复
