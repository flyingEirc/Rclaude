# 测试错误

## 当前状态

- 已清零。

## 失败与修复

### 沙箱缓存目录只读

- 命令：`make lint`、`make test`
- 现象：Go 与 golangci-lint 需要写 `/root/.cache`，默认沙箱中失败。
- 处理：按权限流程重跑对应命令。

### lint 失败

- 命令：`make lint`
- 现象：
  - `errcheck`：多处 `_ =` 忽略错误。
  - `gosec`：PTY spawn 使用配置来源 binary。
  - `errorlint`：错误格式化未包装 error。
  - `gocognit`：测试 harness cleanup 复杂度超限。
  - `govet shadow`：测试 goroutine 中 `err` 变量遮蔽。
- 处理：显式处理或记录关闭/恢复错误，拆分 cleanup helper，补充 `gosec` 说明，修复变量遮蔽。
- 结果：最终 `make lint` 输出 `0 issues.`。

### race/test 失败

- 命令：`make test`
- 现象：
  - `pkg/ptyservice` fake host `Wait` 读取退出信息未加锁，race detector 报告数据竞争。
  - `pkg/syncer` daemon 测试构造的配置缺少 `PTY.FrameMaxBytes`，导致 daemon 提前配置校验失败。
- 处理：fake host `Wait` 加锁读取退出信息；`testDaemonConfig` 补齐 `config.DefaultPTYFrameMaxBytes`。
- 结果：最终 `make test` 全量通过。
