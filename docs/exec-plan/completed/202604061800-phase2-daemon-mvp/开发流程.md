### T7 - `pkg/syncer/daemon.go` + 集成测试
- 新增 daemon 主编排：

  - recv loop 处理 `Read` / `Stat` / `ListDir` 请求

  - heartbeat loop 定时发送心跳
  - 指数退避重连
- 集成测试覆盖：
  - 首次 `file_tree` 上报
  - `Read` / `Stat` / `ListDir` 响应
  - `fsnotify` 变更推送
  - 心跳发送
  - 临时拨号失败后的重连

## 偏离说明

- `mingw32-make fmt` 在当前 Windows/MSYS 环境下因 `bash.exe: couldn't create signal pipe, Win32 error 5` 失败。
- 为保持与 `Makefile` 同等语义，本阶段改为直接执行 `gofumpt` 与 `gci`。
- `lint` / `test` / `build` / `proto` 也统一直接执行底层命令，避免同一 MSYS 层问题影响验收。
