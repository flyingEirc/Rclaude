# Phase 7 — 测试错误

## 2026-04-08

- `make lint` 在 `/workspace/localdemo` 中失败：
  - 现象：`golangci-lint` 报 `can't get config directory`，底层是 `getwd: no such file or directory`
  - 影响：无法直接在 FUSE 挂载目录内完成标准 `make lint`
  - 已尝试：
    - 直接执行 `golangci-lint run --config /workspace/localdemo/.golangci.yml ./...`
    - 从稳定本地目录执行 `golangci-lint run --config /workspace/localdemo/.golangci.yml /workspace/localdemo/...`
    - 使用 `GOFLAGS=-C=...` 变通
  - 结果：均因 FUSE 工作目录解析限制失败
  - 当前结论：这是挂载环境限制，不是本阶段脚本或 Go 代码错误；如需完整 lint，只能在非 FUSE checkout 中执行
  - 后续跟踪：见 [../202604081745-fuse-cwd-module-discovery-compat/plan.md](/workspace/localdemo/docs/exec-plan/active/202604081745-fuse-cwd-module-discovery-compat/plan.md)
